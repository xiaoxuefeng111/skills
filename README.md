# Skills 技能仓库

Claude Code 技能集合。

## 技能列表

### appium-flow-audit

Appium 自动化流程审核与脚本生成技能。

**功能：**
- 双模式生成：自动生成 + 手动生成
- 智能匹配定位器
- 定位器验证
- Fallback 坐标定位
- PopupWindow 兼容

**安装：**
```bash
npx skills add https://github.com/xiaoxuefeng111/skills --skill appium-flow-audit
```

**使用：**
- `录制脚本`
- `采集页面`
- `生成脚本`
- `流程审核`

## 添加新技能

1. 在根目录创建技能文件夹

3. 更新本 README.md 的技能列表
